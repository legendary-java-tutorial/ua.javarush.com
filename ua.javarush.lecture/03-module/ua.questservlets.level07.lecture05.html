6.1 Різні способи оголошення функцій

Ще трохи корисної інформації про функції JavaScript. Функції можна оголосити кількома способами, кожен з них має свої нюанси.

Найстандартніший спосіб такий: ключове слово functionта ім'я.

   function print(data)
   {
     console.log(data);
   }

Другий спосіб це спочатку оголосити змінну, а потім привласнити їй анонімну функцію.

window.print = function(data)
 {
     console.log(data);
 }

Ці два способи дають абсолютно еквівалентний результат . Коли ви оголошуєте звичайну функцію першим способом, то об'єкт window створює нове поле з ім'ям твоєї функції і в нього присвоюється посилання на неї.

6.2 Анонімні функції

Також можна створити анонімну функцію і не надавати її значення нічого. А навіщо така функція потрібна? Як її спричинити?

А вся річ у тому, що можна викликати її одразу. Допустимо ми оголосабо функцію tempі відразу викликали її:


var temp = function(data)
    {
        console.log(data);
    }
 
temp("some info");

Також можна її оголосити і відразу викликати:


   (function(data)
    {
        console.log(data);
    })("some info");

Чимось нагадує анонімні inner-класи в Java.

6.3 Метод eval()

І ще один цікавий спосіб виконати код JavaScript - це взагалі не створювати функції. JavaScript можна просто виконати код, заданий у вигляді рядка. І тому є спеціальна функція eval()(від evaluation). Загальний формат виклику виглядає так:

var result = eval("код або вираз");

Приклади:


var x = eval("1/2");
eval("alert('Hi!')");
