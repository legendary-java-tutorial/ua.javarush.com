4.1 Об'єкти браузера

Як ти вже знаєш, мова JavaScript працює всередині браузера, тому йому якось потрібно взаємодіяти з цим браузером. Для цього браузер має кілька об'єктів і функцій, які доступні з JavaScript.

По-перше, це об'єкт window, що описує вікно браузера. Точніше, раніше описував, коли браузери були без вкладок. Тепер об'єкт window описує поточну вкладку браузера, на яку завантажена сторінка зі скриптом.

По-друге, це об'єкт document, який відповідає за документ, що відображається у вкладці. Це може трохи плутати. Для простоти скажемо, що document – ​​це те, що відображається користувачеві, а window – це те, у чому цей document відображається.

По-третє, це об'єкт console, який відповідає за виведення в консоль браузера. Так, у браузерів теж є консоль, вона використовується переважно для налагодження скрипту та виведення повідомлень про помилки. Зазвичай відкрити її можна, натиснувши кнопку F12 у вашому браузері.

Объект window, об'єкт верхнього рівня для всіх об'єктів сторінки. І хоча ви можете прямо писати імена об'єктів documentу consoleвашому скрипті, насправді їх імена window.documentі window.console.

4.2 Діалоги у JavaScript

Метод alert()

Також об'єкта windowє кілька функцій, які видно скрипту сторінки. Найчастіше використовувана - це функція alert(). Вона дозволяє відобразити діалогове вікно з повідомленням користувача. Робота скрипта при цьому припиняється, доки користувач не натисне OK.

Приклад:

alert("JavaScript is the best!");

Метод prompt()

Також є функція, за допомогою якої можна попросити користувача ввести якесь значення – це prompt().

Приклад:

var age = prompt("Введіть рік народження");

Метод confirm()

Можна вивести діалогове вікно з двома кнопками Ok– Cancelзапитати користувача про якусь дію.

Приклад:

var isOK = confirm("Are you ready");

4.3 Подія onload()

І ще один корисний та цікавий момент. У вікна браузера є event (подія), що виникає, коли документ повністю завантажено. Ти можеш задати код, який повинен виконатися після того, як документ буде завантажено та відображено.

Зробити це можна трьома способами.

По-перше, можна вбудувати JavaScript-код прямо в HTML-сторінку як значення атрибута:

<body onload="alert('документ завантажений');">
  <img src="big-image.png">
</body>

По-друге, можна вбудувати JavaScript-код просто в HTML-сторінку:

<head>
    <script>
        function load()
        {
            alert('документ завантажений');
        }
        window.onload = load;
    </script>
</head>
<body>
    <img src="big-image.png">
</body>

По-третє, можна записати трохи коротше, оголосивши анонімну функцію:

<head>
    <script>
        window.onload = function () {
            alert('документ завантажений');
        }
    </script>
</head>
<body>
    <img src="big-image.png">
</body>

4.4 Подія onclick()

І нарешті, ще одна важлива подія (event) – це event OnClick. Який виникає у будь-якого елемента, на який клацнув користувач. Так само, як і event OnLoadйого можна задати різними способами, тому ми наведемо лише найпростіший:

<body>
  <img src="big-image.png" onclick="alert('користувач клікнув по картинці');">
</body>
