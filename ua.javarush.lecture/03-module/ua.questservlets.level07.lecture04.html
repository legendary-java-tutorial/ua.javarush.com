5.1 document.getElementById()

Як ти знаєш, JavaScript створабо, щоб додати трохи анімації на HTML-сторінки. Поки що ми практично нічого не сказали про це. Як додати цю саму анімацію? Як, наприклад, змінити колір якогось елемента?

Щоб змінити колір елемента, спочатку потрібно знайти цей елемент. Для цього є функція document.getElementById();Ти у твоєму HTML прописуй idу потрібного елемента, а потім звертаєтесь до нього із JavaScript. Приклад:


<head>
    <script>
        function changeImage()
        {
           var image = document.getElementById("image123");
            image.src = "new-image.png";
        }
    </script>
</head>
<body>
    <img id="image123" scr="big-image.png" onclick="changeImage()">
</body>

Після того, як користувач кликає мишкою по картинці, викликається функція changeImge(). У неї всередині ми отримуємо елемент imageза його ID, а потім змінюємо у нього значення src атрибута на нове. Що призведе до того, що елемент imgзавантажить нову картинку.

5.2 document.createElement()

Крім отримання елемента, ми можемо створювати нові елементи і видаляти існуючі.

Створювати елементи дуже легко. Для цього потрібно зробити дві дії: а) створити елемент ; б) додати його в потрібне місце документа. Виглядає це зазвичай приблизно так:


<head>
    <script>
        window.onload = function ()   {
            var image = document.createElement("IMG");
            var image.src = "big-image.png";
            document.body.addAfter(image);
        }
    </script>
</head>
<body>
</body>

5.3 Метод window.navigate()

Є ще один дуже корисний метод – це метод window.navigate(). За його допомогою ти можеш змусити браузер завантажити нову сторінку замість поточної. Просто передай її URLу цей метод і виклич його. Приклад:


  <body> 
    <img scr="big-image.png" onclick="window.navigate('https://google.com');"> 
  </body>

У прикладі вище, після того, як користувач клікне на картинці, в поточну вкладку завантажиться сторінка google.com.
