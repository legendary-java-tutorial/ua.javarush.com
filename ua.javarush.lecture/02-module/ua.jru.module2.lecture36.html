Reflection API. Рефлексія. Темна сторона Java

Рефлексія - це механізм дослідження даних про програму під час її виконання. Рефлексія дозволяє досліджувати інформацію про поля, методи та конструктори класів. Вона використовується практично у всіх сучасних технологіях Java, тому без розуміння її теоретичних та практичних аспектів не обійтися. Для детального знайомства з Reflection API підійде ця стаття.
