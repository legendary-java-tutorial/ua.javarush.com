Java @Аннотації. Що це та як цим користуватися?

Анотації Java - це свого роду мітки в коді, які описують метадані для функції/класу/пакета. Тема складна для новачків, але для твого нинішнього базису знань — якраз.

Ця стаття призначена для людей, які ніколи не працювали з Анотаціями, але хотіли б розібратися, що це та з чим його їдять.

Мінісерія про анотації

  • Анотації. Частина перша, трохи нудна . Тут коротко описані інструкції з типом SOURCE і CLASS. Варто прочитати, щоб не загубитися в другій частині і щоб почати не розуміти трохи більше :)
  • Анотації. Частина друга. Ломбок . У цій частині розповідається про бібліотеку Lombok - відомого представника Source анотацій
