- Привіт, Аміго! Ти вже раніше знайомився з оператором  instanceof . Сьогодні я розповім тобі, як і де його можна використати. instanceof  – це дуже простий та ефективний у використанні оператор.

— Звучить як реклама!

— Він і справді дуже простий. Він використовується у вигляді:  "об'єкт" instanceof "клас".

Він перевіряє, чи є об'єктом певного класу. Все ще простіше, ніж я говорю. Дивись приклад:

Код Опис 
Object o = new Integer(3);
boolean isInt = o instanceof Integer;
 isInt  дорівнюватиме  true . Об'єкт, який посилається змінна o, є об'єктом класу  Integer . 
Object o = "Mama";
boolean isInt = o instanceof Integer;
 isInt  дорівнюватиме  false . Об'єкт, який посилається змінна o, перестав бути об'єктом класу  Integer , є об'єктом класу  String . 
InputStream is = new FileInputStream("");
boolean isFIS = is instanceof
FileInputStream;
 isFIS  дорівнюватиме  true . Об'єкт, який посилається змінна is, є об'єктом класу  FileInputStream .

- Так, дуже просто.

— Цей оператор враховує і спадкування. Ось дивись.

Код Опис 
class Animal
{
}
class Cat extends Animal
{
}
class Tiger extends Cat
{
}
 Тут бачимо три оголошених класу: тварина, кіт і тигр. Кіт успадковується від Тварини. А Тигр від Кота. 
Object o = new Tiger();
boolean isCat = o instanceof Cat;
boolean isTiger = o instanceof Tiger;
boolean isAnimal = o instanceof Animal;
 isCat  дорівнюватиме  true .
isTiger  дорівнюватиме  true .
isAnimal  дорівнюватиме  true .		 
Object o = new Animal();
boolean isCat = o instanceof Cat;
boolean isTiger = o instanceof Tiger;
boolean isAnimal = o instanceof Animal;
 isCat  дорівнюватиме  false .
isTiger  дорівнюватиме  false .
isAnimal  дорівнюватиме  true .

І навіть інтерфейси:

Код Опис 
interface Moveable
{
}
class Cat
{
}
class TomCat extends Cat implements Moveable
{
}
 Створимо два класи: Cat, TomCat та інтерфейс Moveable 
Cat o = new TomCat();
boolean isCat = o instanceof Cat;
boolean isMoveable = o instanceof Moveable;
boolean isTom = o instanceof TomCat;
 isCat  дорівнюватиме  true .
isMoveable  дорівнюватиме  true .
isTom  дорівнюватиме  true .		 
Cat o = new Cat();
boolean isCat = o instanceof Cat;
boolean isMoveable = o instanceof Moveable;
boolean isTom = o instanceof TomCat;
 isCat  дорівнюватиме  true .
isMoveable  буде дорівнює  false .
isTom  буде дорівнює  false .

Оператор instanceof має вигляд:  a  instanceof  B .

Іншими словами, оператор  instanceof  поверне значення  true , якщо:

1)  змінна  а  зберігає посилання об'єкт типу  B

2)  змінна a  зберігає посилання об'єкт, клас якого успадкований від  B

3)  змінна  а  зберігає посилання об'єкт реалізує інтерфейс  B

Інакше оператор  instanceof  поверне значення  false .
