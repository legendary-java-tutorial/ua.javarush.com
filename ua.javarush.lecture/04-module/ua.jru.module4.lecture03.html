Завдання — створити 6 таблиць у трьох схемах, які створено раніше

----------------------------------------

Завдання: дотримуючись інструкцій з шостого уроку п'ятого рівня, зроби таке:

  1. Створи у схемі space таблицю object із п'ятьма колонками:
    • id типу INT
    • name типу VARCHAR(100)
    • type типу VARCHAR(50)
    • discoverer типу VARCHAR(100)
    • discovery_date типу DATE
  2. Створи в схемі library:
    1. таблицю publisher з двома колонками:
      • id типу INT
      • name типу VARCHAR(100)
    2. таблицю author з чотирма колонками:
      • id типу INT
      • first_name типу VARCHAR(50)
      • last_name типу VARCHAR(50)
      • full_name типу VARCHAR(100)
    3. таблицю book із шістьма колонками:
      • id типу INT
      • title типу VARCHAR(100)
      • author_id типу INT
      • publisher_id типу INT
      • publication_year типу INT
      • isbn типу BIGINT
  3. Створи у схемі lego:
    1. таблицю part з чотирма колонками:
      • id типу INT
      • number типу VARCHAR(25)
      • name типу VARCHAR(100)
      • weight типу FLOAT
    2. таблицю lego_set із п'ятьма колонками:
      • id типу INT
      • number типу INT
      • name типу VARCHAR(100)
      • released типу INT
      • inventory типу INT
