У першому рядку ти можеш обрати тип конфігурації. Комп'ютер для розробки (Development Computer) — це те, що нам потрібно.

Також тут пропонується вказати порт, через який можна буде звертатися до MySQL-серверу. А також інсталятор відразу пропонує подбати про твій Firewall, щоб він не блокував цей порт. Більше нам нічого не потрібно.

Крок 3. На наступному кроці інсталятор запропонує створити користувачів, які будуть виконувати запити на цьому SQL-сервері.

Оскільки сервер пропонується лише для навчання та розробки, залишимо тільки одного root-користувача: введи свій пароль. Коли в майбутньому десь знадобиться ввести цей пароль, я говоритиму про нього як про mysql-root-password.

Крок 4. Твій MySQL Server у майбутньому може працювати як звичайна програма (application) або як service.

4. Запуск та зупинка сервісу

Запустити та зупинити MySQL сервер дуже легко. Оскільки ми встановили його як Windows Service, то він у тебе весь час буде запущений.

Якщо тобі потрібно буде зупинити його, натисни Ctrl+Shit+Esc та обери вкладку Services:

За допомогою кліку правою кнопкою мишки на імені сервісу ти завжди можеш його запустити або зупинити, а також переглянути його поточний статус роботи.

5. MySQL Console

І хоча графічного інтерфейсу в MySQL-сервера немає, йому можна віддавати команди, якщо запустити MySQL Console.

Тобі потрібно зайти до папки, куди було встановлено MySQL, у моєму випадку це:

C:\Program Files\MySQL\MySQL Server 8.0\bin

І запустити програму mysql.exe такою командою:

mysql.exe -u root -p

Потім консоль запропонує ввести пароль твого root-користувача. (mysql-root-password):
