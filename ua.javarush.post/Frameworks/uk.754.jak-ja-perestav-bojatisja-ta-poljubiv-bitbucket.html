----------------------------------------

Розглянемо, як можна використовувати сервіс BitBucket для збереження коду завдань. " Навіщо?! - Запитайте ви. - Адже в Intellij IDEA є вбудований доступ до GitHub! " Дві причини. Перша : GitHub на безкоштовному обліковому записі дозволяє зберігати тільки відкриті проекти, а перше правило CodeGym - нікому не говори про CodeGym Чи можна давати комусь рішення задачі? Ні, давати готове робоче рішення завдання заборонено. Може призвести до лазні на сайті. Друга : Mercurial vs git. Але тут, звичайно, кому як зручніше. Тому в кілька простих кроків ми налаштуємо BitBucket для використання в наших цілях (зберігати, і, можливо, розгалужувати код).

  1. Реєструємось на BitBucket

    Зауважте, можна використовувати і ваш Google-аккаунт!

  2. Встановлюємо плагін для роботи з репозиторієм:

    Заходимо в налаштування IDEA/plugins тиснемо «Переглянути репозиторії плагінів»

    Шукаємо «bitbucket» та встановлюємо

    IDEA попросить перезавантаження...

    За допомогою плагіна можна керувати репозиторієм прямо з IDEA

    Для коректної роботи вам знадобиться встановлений в системі Mercurial .

В Ubuntu це просто: sudo apt-get install mercurial хоча я рекомендую спочатку додати ppa з найсвіжішими версіями mercurial-ppa : sudo apt-add-repository ppa:mercurial-ppa/releases для нього знадобиться публічний ключ: sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-key 323293EE після ви можете sudo apt-get install mercurial Для товаришів з windows доведеться качати установник з офіційного сайту. Після цього йдемо VCS/Import to version control/Share project (поруч із написом побачите симпатичне відерце). Запускаємо! Плагін попросить авторизуватися на BitBucket (введіть логін та пароль).
