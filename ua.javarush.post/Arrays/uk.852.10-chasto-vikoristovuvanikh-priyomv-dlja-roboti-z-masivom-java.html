10 часто використовуваних прийомів для роботи з масивом Java

----------------------------------------

Наступні 10 прийомів для часто використовуються. Вони мають найвищий рейтинг у Stack Overflow – системі питань та відповідей про програмування.

----------------------------------------

Наступні 10 прийомів для масивів в Java часто використовуються. Вони мають найвищий рейтинг у Stack Overflow – системі питань та відповідей про програмування.

  1. Оголошення масиву

    String[] aArray = new String[5];
String[] bArray = {"a","b","c", "d", "e"};
String[] cArray = new String[]{"a","b","c","d","e"};

  2. Виведення масиву в Java

    int[] intArray = { 1, 2, 3, 4, 5 };
String intArrayString = Arrays.toString(intArray);

// print directly will print reference value
System.out.println(intArray);
// [I@7150bd4d

System.out.println(intArrayString);
// [1, 2, 3, 4, 5]

  3. Створення ArrayList із масиву

    String[] stringArray = { "a", "b", "c", "d", "e" };
ArrayList<String> arrayList = new ArrayList<String>(Arrays.asList(stringArray));
System.out.println(arrayList);
// [a, b, c, d, e]

  4. Перевірка масиву на наявність конкретного значення

    String[] stringArray = { "a", "b", "c", "d", "e" };
boolean b = Arrays.asList(stringArray).contains("a");
System.out.println(b);
// true

  5. Об'єднання двох масивів

    int[] intArray = { 1, 2, 3, 4, 5 };
int[] intArray2 = { 6, 7, 8, 9, 10 };
// Apache Commons Lang library
int[] combinedIntArray = ArrayUtils.addAll(intArray, intArray2);

  6. Оголошення масиву в одному рядку

    method(new String[]{"a", "b", "c", "d", "e"});

  7. Об'єднання елементів масиву в рядок

    // containing the provided list of elements
// Apache common lang
String j = StringUtils.join(new String[] { "a", "b", "c" }, ", ");
System.out.println(j);
// a, b, c

  8. Перетворення ArrayListна масив

    String[] stringArray = { "a", "b", "c", "d", "e" };
ArrayList<String> arrayList = new ArrayList<String>(Arrays.asList(stringArray));
String[] stringArr = new String[arrayList.size()];
arrayList.toArray(stringArr);
for (String s : stringArr)
	System.out.println(s);

  9. Перетворення масиву на безліч ( Set)

    Set<String> set = new HashSet<String>(Arrays.asList(stringArray));
System.out.println(set);
//[d, e, b, c, a]

  10. Повернення масиву з елементами у зворотному порядку

    int[] intArray = { 1, 2, 3, 4, 5 };
ArrayUtils.reverse(intArray);
System.out.println(Arrays.toString(intArray));
//[5, 4, 3, 2, 1]

  11. Видалення елемента з масиву

    int[] intArray = { 1, 2, 3, 4, 5 };
int[] removed = ArrayUtils.removeElement(intArray, 3);//create a new array
System.out.println(Arrays.toString(removed));
