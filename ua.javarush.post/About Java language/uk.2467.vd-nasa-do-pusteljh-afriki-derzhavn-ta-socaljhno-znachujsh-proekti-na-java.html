Електронна система охорони здоров'я в Австрії

Австрія відома своєю ультрасучасною та зручною системою соціального забезпечення, яка включає чудову охорону здоров'я, надійне соціальне страхування, а також велику мережу лікарень, лікарів та аптек. Австрійський закон про соціальне забезпечення, система державного страхування, що включає 22 установи з різними видами страхового покриття та соціальні програми, страхує більшість австрійських громадян. А останніми роками до цієї системи додали ще й одну з найпросунутіших у світі програм для смарт-карт та електронних медичних записів, яка працює значною мірою завдяки Java. “Ми обрали Java через основні переваги цієї мови, насамперед її величезної універсальності та гнучкості серед процесів та апаратних платформ,” – зазначив Райнер Шюгерль (Rainer Schügerl), директор з розробки програмного забезпечення та безпеки австрійської організації SVC, яка займається інноваційними рішеннями в галузі телематики у охороні здоров'я та електронного уряду. У рамках цієї системи всі застраховані громадяни отримують смарт-картку, яка перевіряє їх статус страхування та полегшує створення, передачу та зберігання електронних медичних карток. Система електронних карток також дозволяє громадянам користуватися різними послугами електронної охорони здоров'я, від профілактичних оглядів до програм лікування захворювань. Тисячі постачальників медичних послуг встановабо спеціальне обладнання для сканування смарт-карток та передачі особистих даних через захищену інформаційну мережу охорони здоров'я, яка не тільки охоплює Австрію, але й поширюється на багато інших європейських країн, будучи частиною проекту NETC@RDS Європейського Союзу. Саме Java - визначальна технологія для цієї системи. “Java – це стабільна, високоякісна мова програмування, яка відповідає всім нашим потребам. Для розробок корпоративного рівня, що вимагають високої доступності, надійності та безпеки, більшість австрійських організацій застосовують саме Java», - додав Райнер Шюгерль. будучи частиною проекту NETC@RDS Європейського Союзу. Забезпечення чистою питною водою жителів Африки

Спеціальний мобільний додаток, написаний на Java, збирає та впорядковує результати роботи кенійського державного проекту Safe Water Kenya, завдання якого забезпечити чистою питною водою сільські сім'ї у віддалених регіонах Східної Африки. Згідно з інформацією Всесвітньої організації охорони здоров'я, понад 2 мільйони людей, 95% з яких діти щорічно помирають від наслідків різних захворювань, що передаються через воду. "Окрім шкоди для здоров'я, це чинить і негативний економічний вплив на бідні верстви населення Африки, адже під час хвороби вони не можуть працювати і тим самим заробляти на життя", - зазначає Дон Арнольд (Don Arnold), виконавчий директор та засновник організації Safe Water. Кенія (SWK). Щоб вирішити критичні проблеми із чистою питною водою у сільських районах Кенії, SWK почала встановлювати фільтри для води Hydraid BioSand, створені на основі екологічної технології фільтрації за допомогою піску. “Ми встановабо вже понад 2500 систем фільтрації води по всій Кенії. За нашими підрахунками, в середній кенійській сім'ї семеро людей, тому всього за рік або близько того ми покращабо життя більш ніж сімнадцяти тисяч людей. Це досить складний процес. Щоб звітувати перед нашими донорами про встановлення кожного фільтра, нам потрібно надавати безліч інформації: фотографії, GPS координати та підписи сімей, які отримали фільтр», - сказав Дон Арнольд. Для цієї мети і застосовується рішення на базі Java - програма під назвою Survey App, призначена для пристроїв на базі мобільної операційної системи Android. Ця програма була розроблена американською компанією mFrontiers, яка в 2014 році отримала за нього премію Oracle Excellence Award - за інновації в галузі сталого розвитку. За допомогою цієї програми після встановлення кожного фільтра працівники SWK додають до системи інформацію про нього. “Як правило, вони заповнюють опитувальник, який складається із шести-восьми сторінок на планшеті Android, де на кожній сторінці представлено п'ять чи шість питань. За допомогою того ж планшета із встановленим на ньому додатком вони можуть робити фотографії фільтра та сім'ї, яка його отримала, а також додавати координати місця, тому що зазвичай у таких місцях немає точних адреса,” – розповів Деніел Панг (Daniel Pahng), голова компанії mFrontiers, який розробив програму Survey App самостійно за допомогою відкритої JavaScript бібліотеки jQuery. На останньому етапі програма зберігає підпис одержувача та серійний номер фільтра води. Оскільки у віддалених районах Кенії доступу до Інтернету зазвичай немає, отримані дані зберігаються на Android планшеті в базі даних Oracle Berkeley DB. Коли ж представники Safe Water Kenya повертаються до штаб-квартири організації, дані автоматично завантажуються з планшета в хмарну базу даних.

Дослідження космосу NASA

Американське агентство NASA використовує Java в низці досить цікавих проектів і додатків. Одним із таких проектів є World Wind. Це SDK, за допомогою якого можна зменшувати масштаб зображень Землі, зроблених із супутників. За допомогою даного інструменту всі бажаючі можуть досліджувати різні ділянки поверхні планети. Джерелом даних для такої моделі є комбінація супутникових знімків Landsat та даних топографічної місії Shuttle Radar. Інженери NASA демонструють більш ніж 90 прикладівтого, як можна використовувати цей SDK. В цілому ж, Java-технології використовуються NASA в різних проектах в рамках космічної програми і досліджень космосу. “Досі продуктивність Java ніколи нас не підводила. Ми справді задоволені показниками розроблених нами Java-додатків. Жодна з проблем продуктивності, з якими ми стикалися досі, не була пов'язана з вибором Java як платформа. Більшість наших складнощів пов'язані з обмеженнями обсягів даних і швидкістю застарілого програмного забезпечення. Однією з унікальних вимог є те, що ми повинні використовувати акредитоване програмне забезпечення. Ми не можемо просто витягувати сторонні JAR-файли за бажанням, але ми можемо використовувати все, що вже міститься в JDK, - розповів Нік Себі (Nick Sabey), старший аналітик та розробник програмного забезпечення у групі NASA з оцінки ризиків для роботизованих з'єднань. До речі, NASA продовжує активно наймати Java-розробників.Вакансії американського космічного агентства для Java-кодерів можна переглянути, наприклад, на спеціальній сторінці на сайті Indeed.

Віртуальний огляд пацієнтів лікарем

Ще одна система з великим потенціалом для застосування в галузі охорони здоров'я, особливо в країнах, що розвиваються - Online Doctor System , програми якої працюють на Java. Ця система допомагає лікарям проводити віртуальні огляди та консультації пацієнтів онлайн. Програма дозволяє пацієнту заповнити спеціальну форму за кілька секунд до візуального огляду лікарем. У нього також можна завантажувати результати обстежень (наприклад, рентгенівські знімки), історію хвороби та багато іншого. Додаток Online Doctor System включає цілий ряд модулів, щоб спростити проведення медичного огляду лікарем. Пацієнти можуть обирати лікарів, що їх цікавлять, і записуватися до них на прийом, оплачуючи ці прийоми також через додаток банківською картою. Як зазначають розробники, у майбутньому технології Java можна буде використовувати і для створення додатків-докторів, де огляд пацієнтів проводитиме штучний інтелект. Він видаватиме діагноз та рекомендоване лікування, спираючись на масив зібраних даних про пацієнтів, а також завантажені ними результати аналізів та іншу інформацію. Роботи-доктора, інтегровані в різні пристрої, вже скоро зможуть виконувати дуже багато нескладних процедур, таких як вимірювання температури пацієнта, частоти його серцебиття або тиску. Експерти зазначають, що дані технології значно здешевлять кваліфіковану медичну допомогу, зробивши її доступною навіть для найменш забезпечених верств суспільства.

Автономний транспорт

Java активно використовується і в проектах "розумного", тобто автономного, транспорту. Зокрема, компанія Perrone Robotics розробила на основі Java цілу низку програмних компонентів для роботизованого транспорту. Perrone Robotics має цілу платформу рішень для автономних транспортних засобів під назвою MAX, і багато її компонентів написані на Java. Ця платформа застосовується в різних сферах: від автоматизованих шатлів і автобусів до великих індустріальних вантажівок і будівельної техніки. Як відзначають розробники системи, такої універсальності своєї системи їм вдалося досягти, зокрема, завдяки застосуванню Java. Наша система розроблена так, щоб підходити для всіх типів транспортних засобів. Те саме прикладне програмне забезпечення працює з широким спектром платформ і додатків. Реалізувати це вдалося завдяки тому, що наша система складається з набору програмних будівельних блоків, які можна застосовувати для найширшого діапазону транспортних засобів і робототехніки.
