Зустрічається в одному із завдань (не на Javarush): Є завдання – невелика програма на Java. Одна з умов: запуск програми через public static void main (String [] args), args [] параметра: файл вхідних даних, файл для запису відповіді. Не можу збагнути, як у args[] передати імена файлів: input.txt, jutput.txt. Підкажіть, будь ласка, якщо можна – на прикладі.
