Щоб отримати уявлення про те, як IntelliJ IDEA допоможе вам у розробці та запуску Java-додатків, ми пропонуємо вам створити і запустити найпростіший приклад Hello, World в даній програмі. Таким чином, ви зможете дізнатися про основні функції IDE без необхідності вдаватися в деталі коду. Покрокова інструкція допоможе вам не заплутатися в тонкощі запуску та налаштування програми.

Перед початком роботи

Для створення Java-додатків потрібно Java Development Kit ( JDK ). Якщо програма ще не встановлена, її можна завантажити та встановити з сайту Oracle.com . Там же можна знайти інструкції щодо встановлення.

Створення проекту

Створення будь-якої програми в IntelliJ IDEA починається зі створення проекту (дізнатися, чому проект необхідний, можна в IntelliJ IDEA Help, за посиланням Project ), так що першим нашим кроком буде створення проекту Hello, World. Цей проект міститиме модуль Java для нашої програми Java.

  1. Якщо жоден проект наразі не відкритий, натисніть кнопку Create New Project на екрані привітання. В іншому випадку виберіть пункт New Project у меню File. В результаті відкриється майстер створення проекту.

  2. У лівій панелі виберіть Java Module .

  3. У правій частині сторінки в полі Project name введіть назву проекту: HelloWorld.

  4. Якщо до цього ви ніколи не налаштовували JDK в IntelliJ IDEA (у такому випадку в полі Project SDK стоїть <Nоne> ), необхідно зробити це зараз.

    Замість <Nоne> натисніть New і виберіть JDK у підменю .

    У вікні Select Home Directory for JDK виберіть директорію, в яку встановлювався JDK, і натисніть OK .

    Версія JDK, яку ви вибрали, з'явиться у полі Project SDK .

    Клацніть Next .

    Зверніть увагу, що вказана версія JDK буде пов'язана за замовчуванням з усіма проектами та Java модулями, які надалі будуть створюватися.

  5. На наступній сторінці здійснюється вибір майстра для вказівки додаткових технологій, які будуть підтримуватись у нашому модулі.

    Оскільки наш додаток буде "старим добрим додатком Java", ми не потребуємо будь-якої з цих технологій. Так що просто натисніть кнопку Finish .

    Зачекайте, доки IntelliJ IDEA створює необхідні структури проекту. Коли цей процес завершиться, ви можете побачити структуру нового проекту у вікні Project .

Вивчення структури проекту

Давайте пробіжимо поглядом по структурі проекту. У дереві проекту бачимо дві директорії верхнього рівня:

  • HelloWorld . Це вузол, що містить ваш Java модуль. Папки .idea та файли всередині директорії HelloWorld.iml використовуються для зберігання даних конфігурації вашого проекту та модулів відповідно. SRC папки містять вихідний код.

  • External Libraries (Зовнішні бібліотеки). Це категорія, яка представляє всі зовнішні ресурси, необхідні для вашого проекту. В даний час у цій категорії файли .jar, з вибраного нами JDK.
З усіх згаданих папок у прикладі нам знадобиться тільки SRC .

Створення пакету

Тепер ми збираємося створити пакет для HelloWorld (Ми створимо цей клас трохи пізніше.) Назвемо цей пакет com.example.helloworld.

  1. У вікні інструментів Project виберіть папку SRC та натисніть ALT + INSERT. (В якості альтернативи ви можете вибрати File -> New , або New з контекстного меню для папки SRC).

  2. У меню New виберіть Package . (можна використовувати стрілки вгору та вниз для навігації по меню, ENTER для вибору виділеного елемента)

  3. У вікні New Package, що відкрилося , введіть ім'я пакета (com.example.helloworld). Натисніть кнопку ОК (або клавішу ENTER).

    Новий пакет з'явиться у вікні Project .

Створення класу

  1. Натисніть клавішу ALT + INSERT. У вікні New зі списку доступних дій з виділеним пакетом com.example.helloworld вибираємо Java Class та натискаємо Enter.

  2. У вікні Create New Class у полі Name вводимо ім'я HelloWorld. У полі Kind залишаємо тип Class та натискаємо Enter, щоб підтвердити створення класу.
