Компанія-розробник інструментів Java та інших мов JetBrains провела щорічне дослідження. В опитуванні Developer Ecosystem Survey 2021 року взяли участь понад 47 тисяч людей із 183 країн. У тому числі, під час дослідження опитали айтішників із 17 країн, на які припадає приблизно 70% усіх розробників у світі: Аргентина, Білорусь, Бразилія, Канада, Китай, Франція, Німеччина, Індія, Японія, Мексика, Росія, Південна Корея, Іспанія , Туреччина, Україна, Великобританія та США. Ми зібрали ключові висновки дослідження — про мову Java, роботу та спосіб життя програмістів.

Java в 2021 році: популярна в Південній Кореї, Китаї та Німеччині

Одне з головних висновків дослідження — це те, що мова програмування Java хоч і трохи втрачає позиції, все ж таки популярніша, ніж Python як основна мова (32% проти 29% опитаних). Найпопулярнішою мовою в 2021 році став JavaScript - 69% респондентів використали його за останні 12 місяців, а 39% назвали його основною мовою програмування. Як і в 2020 році, Java найчастіше використовують у Південній Кореї, Китаї та Німеччині. Частка Java у Південній Кореї становить 55%, у Китаї – 47%, а у Німеччині – 33%. У Білорусі 26% розробників віддають перевагу Java, в Україні — 20%, Росії — 18%. Це трохи менше, ніж 2020 року.

Яка версія Java найпопулярніша?

Як і в 2020 році, Java 8, як і раніше, провідна версія цієї мови програмування. Цікаво й те, що з 2019 року Java 11 щорічно демонструє стабільне зростання приблизно на 10 процентних пунктів. У цифрах це виглядає так: 72% розробників, які використовують Java, віддають перевагу Java 8, а 42% - Java 11. Частки Java 12 і Java 13 знизабося приблизно втричі з 2020 року, тоді як новіші Java 14 і Java 15 вже отримали аудиторію, незважаючи на те, що вони відносно нові.

Які фреймворки вибирають

Частка розробників, які використовують Spring Boot, трохи зросла у 2021 році, приблизно на 4 відсоткові пункти (його обирають 65% програмістів). На другому місці фреймворк Spring MVC – 42% розробників. У той же час, невелика частина розробників використовує набагато більше нових веб-фреймворків, ніж у 2020 році.

Сервери додатків: Apache Tomcat та Jetty

Ландшафт серверів додатків у 2021 році приблизно такий самий, як і у 2020 році, Apache Tomcat, як і раніше, найбільш популярний серед розробників — його використовують 61% програмістів.

Для чого найчастіше використовують Java?

Найчастіше Java використовують для розробки веб-сайтів (39%), рідше - для створення утиліт (26%) та системного ПЗ (19%). Використання Java для веб-сайтів та ІТ-інфраструктури незначно зросло в період з 2020 по 2021 рік, в той час як Java почала менше використовуватися для бізнес-аналітики, обробки даних та машинного навчання.

Портрет розробника: у галузь приходить більше жінок

63% девелоперів працюють на фултаймі, 15% - навчаються у вузах, 7% - також навчаються, але працюють, 5% - фрілансят. 81% респондентів працюють на позиції програміста, 15% - тимліда, 13% - DevOps-інженера, 12% - архітектора. Більшість програмістів працюють у компаніях, де розробників менше, ніж фахівців з інших областей. Ще один примітний висновок — це те, що найчастіше розробники працюють у компаніях чисельністю не більше ніж 50 осіб (так відповіли 80% опитаних). Якщо говорити про IT-компанії, то найчастіше програмісти працюють з IT-сервісами (40%), мобільною розробкою (24%), хмарними платформами та даними (по 17%). Серед компаній з інших сфер айтішники найчастіше працюють у сфері банкінгу та фінансів (18%), продажу та освіти (по 12%). Основну частку серед розробників займають чоловіки (93%) від 21 до 29 років (48%). Проте все більше жінок приходять у технологічну галузь. Вони частіше працюють з аналізом даних та машинним навчанням, UX/UI-дизайном. Набагато менше жінок працюють із програмуванням та системним адмініструванням. Найбільший відсоток програмісток у Південній Кореї (13%), Білорусі (12%) та Канаді (8%). У Росії у загальному полі розробників лише 7% програмісток, а в Україні – 6%.

Спосіб життя розробника: час пробудження та видалення

Час, коли розробники зазвичай починають свій робочий день, не змінилося з минулого року: програмісти дотримувалися свого робочого графіка навіть під час пандемії. 83% розробників розпочинають свій робочий день до 10 години ранку. Якщо до пандемії близько 70% розробників працювали з офісу, нині 80% працюють із дому. Цікаво, що хоча під час пандемії частка розробників, які працюють у кафе, скоротилася вдвічі, кількість людей, які працюють у коворкінгах, не дуже змінилася. Цього року різноманітні заходи стали набагато доступнішими завдяки широкому поширенню онлайн-форматів. У 2020 році близько 75% розробників відвідали онлайн-конференції, а понад 40% респондентів відвідали онлайн-семінари та зустрічі. Хоча в більшості розробників є обліковий запис GitHub, трохи менше половини мають профіль на Stack Overflow. Три найчастіше використовувані соціальні мережі — це Twitter, Instagram і Facebook.

Хобі розробників: відеоігри випереджають кодинг

У 2020 році більшість розробників назвали програмування своїм найбільшим хобі. Цього року відеоігри очолюють список, їхня популярність зросла з 49% до 59%. Персональний комп'ютер – найпопулярніша ігрова платформа, консолі посіли друге місце. Серед них найбільш широко використовуються PlayStation та Nintendo Switch. На другому місці за популярністю серед хобі – програмування (57%), на третьому – прослуховування музики (40%). Більше половини опитаних займаються благодійністю. Найпопулярніша причина благодійних пожертв – це бажання зробити мир і спільноту зручнішими для життя. Більшість розробників бажають допомогти дітям, піклуються про навколишнє середовище, борються із соціальною нерівністю чи допомагають знедоленим.
