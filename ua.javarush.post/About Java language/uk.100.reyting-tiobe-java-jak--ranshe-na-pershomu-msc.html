----------------------------------------

Відповідно до індексу TIOBE , що оцінює популярність мов програмування на основі підрахунку результатів пошукових запитів, у вересні Java як і раніше утримує пальму першості. Результат нашої улюбленої мови — 12.6% від загальної кількості запитів. Більш ніж на 5% відстає найближчий переслідувач — «старійшина» програмістського світу Сі, а його молодший за віком, але дуже потужний об'єктно-орієнтований братик, С++, закріпився на третьому місці. П'ятірку лідерів замикають C# та Python. Останньому вже котрий рік пророкують швидке лідерство, але не всі аналітики згодні з цією думкою.
