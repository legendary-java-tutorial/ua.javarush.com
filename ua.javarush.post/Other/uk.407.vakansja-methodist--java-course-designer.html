----------------------------------------

Чи любите виступати на конференціях з доповідями з Java? Вчіть людей, ведете навчальні курси з Java або хочете створити свої? Або, можливо, пишіть книги про… Java? Тоді, можливо, вас зацікавить можливість приєднатися до CodeGym. Нам потрібен дуже хороший методист Java.
Що ми пропонуємо:
  • Хороша можливість для самореалізації та професійного зростання;
  • адекватний менеджмент та цікаві завдання;
  • офіс з усіма зручностями (зона відпочинку, бібліотека, кухня, безкоштовний чай, кава, фрукти та печиво);
  • оплачувана відпустка (24 календарні дні на рік), оплачувані лікарняні, медстраховка;
  • зручне розташування офісу (ст.м. Либідська);
  • дружня команда професіоналів;
  • тім-білдінги та корпоративи за рахунок компанії;
  • можливість іноді працювати з дому.
Чим треба буде займатися:
  • формування різних навчальних курсів для різних сегментів користувачів;
  • детальне проектування курсу під кожен сегмент з урахуванням його особливостей;
  • складання плану/сценарію для кожної лекції;
  • складання плану/сценарію для навчальних відео;
  • складання ТЗ для завдань;
  • складання списку питань (та відповідей) для тестування студентів;
  • складання питань для екзаменування студентів;
  • складання питань (і відповідей) для підготовки до співбесіди;
  • розробляти бачення та стратегію розвитку навчальної частини проекту;
  • створювати, змінювати та розвивати концепцію всіх навчальних програм.
Вимоги до кандидата:
  • Мінімум 3 роки досвіду Java-програмістом;
  • мінімум 1 року досвіду викладання та навчання Java;
  • відмінне знання Java Core;
  • досвід роботи з Spring, Hibernate, Servlets;
  • досвід написання навчальних статей або книг.
Буде плюсом:
  • Досвід керування командою;
  • написання книг із програмування;
  • ведення власних навчальних курсів.
Портрет ідеального кандидата: Досвідчений Java-програміст з аналітичним складом розуму, ініціативний і відповідальний, якому подобається навчати людей і допомагати їм знайти своє місце в житті, знайти улюблену професію. Якщо ви отримуєте задоволення навчаючи людей, вам подобається розвиватися і самореалізовуватися у сфері навчання програмуванню, ви любите Java і хочете, щоб його знало якнайбільше людей - надсилайте резюме на пошту: hr@codegym.cc . У темі листа вкажіть: "Java Methodist" .
