У попередній статті ми розповіли про кілька цікавих YouTube каналів, які публікують відеоуроки з програмування для початківців російською мовою. У цій статті мова піде про найцікавіші англомовні ресурси. Якщо ваш рівень англійської є середнім або вищим, рекомендуємо хоча б іноді заходити на ці канали. Заодно підтягнете технічну англійську, вона вам знадобиться. Зрозуміло, всі ці ролики лише доповнення до навчання. Головне – не забувати програмувати щодня!

Derek Banas

За час свого існування канал Derek Banas набрав понад 70 мільйонів переглядів та 764 тисячі передплатників. Популярність заслужена: на каналі викладені ролики на тему мови програмування для початківців, відеоуроки про різні технології. Їхній творець – Дерек Банас – природжений кодер, а, крім того, вміє дуже виразно пояснювати складні речі. Тож якщо знаєте мову, спробуйте їх подивитися. Там є матеріали C++, використання Git, розробці під Android і, зрозуміло, Java. На цьому каналі нас найбільше цікавить програмування на Java для початківців , відеоуроки з об'єктно-орієнтованого дизайну (настійно рекомендую!), а також розділ Java Algorithms . Автор регулярно викладає свій код та коментує посібники на сайтіNewthinktank .

Thenewboston

Ймовірно, один із найбільших ресурсів із навчальними роликами у світі: їх тут близько п'яти тисяч, передплатників — 1.8 млн. та 382 млн. переглядів. Не «Містер Макс», звичайно, але теж результат (про всяк випадок повідомляю, це була гірка іронія). Якщо вам у принципі цікава тема «мови програмування для початківців», види уроків thenewboston - те, що потрібно. Роликів тут кілька тисяч, причому весь контент — на дуже високому рівні. Як і на попередньому каналі, тут можна знайти відеоуроки програмування для початківців на різні теми, від JavaScript до Unreal, а також багато матеріалів за темами, не пов'язаними з програмуванням. Уроки біології чи геометрії, наприклад. По суті, перед нами шкільно-університетський універсальний ресурс із дуже якісним контентом. Нас, зрозуміло, найбільше хвилює Java, і її у надлишку.для новачків та продовжуючих . А, наприклад, у цьому розділі ви знайдете відеоуроки Java для початківців (покрокова розробка гри з графічним інтерфейсом).

Cave of Programming

Ще один канал, який містить відеоуроки програмування для початківців, Cave of Programming – це не лише збірка роликів, а й онлайн-курси (платні та безкоштовні). Інструктор Джон Парселл працював програмістом у низці великих компаній понад 14 років, поки, нарешті, не вирішив, що його покликання навчати людей. Ласкаво просимо до печери Джона, якщо хочете вивчати Java, Android, підтягнути SQL чи C++. Крім «Java для початківців» , на каналі ви знайдете плейлисти по колекціях , багатопоточності та Java 8 . Особливо хочеться відзначити дуже виразний курс з мобільної розробки «Android Java» .

Java Brains

Великий відеоканал, присвячений нашій з вами улюбленій мові та супутнім технологіям, а також JavaScript. Варто звернути увагу на цілу низку курсів з технологій Java EE , а також Spring , базовий курс з лямбда-виразів Java 8, Maven і Hibernate . Як ви зрозуміли, це переважно теми, які вам будуть потрібні не на самому старті навчання. Вони можуть бути корисними тим, хто вже пройшов хоча б половину рівнів онлайн-курсу CodeGym .

Java Tutorial for Beginners

На відміну від попереднього каналу, Java Tutorial for Beginners доброзичливий до новачків. Тут можна знайти ролики по топіку «програмування на Java з нуля», відео, що охоплюють практично всі теми Java Core - синтаксис, ООП, потоки, колекції, сервлети, але також - про шаблони проектування, SOA, веб-сервіси і фреймворки Spring і Hibernate.

Що далі?

Всі люди різні. Хтось надасть перевагу, щоб вивчити програмування з нуля, самовчитель — відео, хтось — навпаки, горою за книги. Моя щира думка: не концентруйтеся на чомусь одному, виберіть кілька джерел і розвивайтеся, доки можете. Ну а якщо знайшли якийсь цікавий канал, на якому викладено цікаві відеоуроки програмування з нуля поза нашим списком — кидайте посилання у коментарі!
