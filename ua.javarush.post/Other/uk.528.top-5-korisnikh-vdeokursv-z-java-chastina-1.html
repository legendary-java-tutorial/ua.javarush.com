Якщо ви вже передплатабо онлайн-курс програмуванняCodeGym, для вивчення Java з нуля відеокурс не надто й потрібен… Чи все-таки потрібен? Я дотримуюсь думки, що добрих джерел має бути багато! Сприйняття у всіх різне, і комусь підійде одне пояснення, а комусь зовсім інше. У цьому топіку зібрані посилання на кілька вдалих російськомовних YouTube-каналів, які містять якісні відеоуроки з Java для програмістів-початківців (від початку навчання до рівня Junior).

Golovach Courses

Ймовірно, найвідоміші відеоуроки з Java російською мовою. Іван Головач, розробник та викладач програмування з Харкова, виклав на своєму YouTube-каналі записи курсів, які він вів у різний час. Тут ви знайдете дуже багато корисного матеріалу з Java Core, багатопотокового програмування, роботи з базами даних в Java-середовищі (JDBC, реляційна алгебра) ... і багато іншого. Сотні годин відео, якщо не тисячі. Особливості інформації дуже багато, викладач пише на дошці маркерами, місцями є повтори, але виклад розумний і професійний. Багато тем Java на російській YouTube можна знайти тільки тут. Якщо ви зараз навчаєте потоки на CodeGym і вони ніяк не йдуть, заходьте до Івана на канал, швидше за все теорію він вам розжує.

Онлайн уроки з Java

На каналі Юрія Ткача «Онлайн уроки з Java» також можна знайти безліч матеріалів для початківців. Дуже корисні для вивчення Java відеоуроки заховані у плейлисті Java для тестувальників, інформація подана дуже просто. А закріпити вивчене можна, переглянувши відеоуроки Java практика. Більшість глядачів зазначає, що стиль у автора не надто академічний і дуже зрозумілий. Тобто засипати, як часто буває на реальних заняттях в університетах, ви не будете. Головне - стежте за перебігом думки. Якщо вам сподобалося, як Юрій веде лекції, ви вже просунулися у вивченні Java Core і готові приступити до фреймворку Spring, спробуйте оцінити найновіші на сьогоднішній день ролики каналу Spring Framework – the Basics. Тут, щоправда, вже не лекції перед аудиторією, як у випадку з Java практика і Java для тестувальників, а спеціально зняті навчальні відео.

Yakov Fain відеокурси з Java

Багато надзвичайно корисних і ґрунтовних уроків для тих, хто вивчає програмування, Java-відео від визнаного ІТ-викладача Якова Файна. Навіть не просто ІТ-викладача, а справжнісінького Java-чемпіона (це спеціальне звання, носять його лише деякі фахівці з мови) та співзасновника компаній Farata Systems і SuranceBay. Зверніть увагу: на каналі є відео і англійською, і російською (Яков Файн вже багато років живе в США). І, головне, автор уміє пояснювати матеріал на різних рівнях. Скажімо, у нього є книга «Програмування на Java для дітей, батьків, бабусь та дідусів», розрахована насамперед на дитячу аудиторію.

javaRussia

Цей канал не такий розкручений, як ті, що ви бачабо вище, але однозначно варто згадувати. Автор - Наїль Алішев - склав дуже непогані короткі відеоуроки програмування Java, в яких максимально доступно викладено базові принципи мови. Роликів не дуже багато, близько 50, тривалість кожного з них – від 7 хвабон до півгодини. Це - самі основи (ідентичні квестам Java Syntax і Java Core), викладені лаконічно і зрозуміло, наведені непогані приклади, зрозумілі новачкам. Теми, що зачіпаються: синтаксис, ООП, установка робочого оточення, виключення, параметризація, клас Object, вкладені та абстрактні класи, серіалізація. А якийсь час тому автор створив уже платний відеокурс Java на платформі Udemy, В якому докладно пояснює такі теми як Java Collections, регулярні висловлювання, багатопоточність і лямбда-вирази.

Кладезь цікавої інформації від Roman Brovko

Мені складно сказати, наскільки «законними» є добірки курсів, зібрані на цьому каналі. Вони взяті із різних курсів (наприклад, Stepic). Тим не менш, помістю їх тут — я знаходила в них багато корисного, і щиро сподіваюся, що канал не прикриють. Роман зібрав на своєму каналі курси для початківців, підготовлені спеціально для онлайн-навчання, а також записи з «живих» лекцій та виступів. Теми: основи Java, ООП, GUI, патерни, Java EE, Spring, Java + JDBC + Hibernate + Spring Data JPA, розробка веб-сервісу Java і навіть мікросервісу.

Що далі?

Багато розробників переконалися власним досвідом: вивчити мову програмування Java відеоуроки з Youtube (особливо якісні) допоможуть. Тільки ось практику ніколи не замінять. Так що комбінуйте джерела, як завгодно, але не забувайте вирішувати практичні завдання , бажано щодня! У наступній частині статті я розповім про корисні англомовні канали, які містять уроки по java.
