Вітаю всіх любителів болю та страждань. Опинившись на порозі Ентерпрайза – залиш надію кожен, хто сюди входить. Сьогодні пропоную вашій увазі невелику добірку книг з Java EE. Але для початку хвабона тиші для Java EE ...
Король помер, нехай живе король! Справа в тому, що в середині серпня 2017 року компанія Oracle виявилася від прав на Java EE на користь Eclipse Foundation, а вже на початку 2018 року за підсумками голосування щодо зміни імені проекту (у зв'язку з тим фактом, що Oracle відмовилася передати Eclipse права на використання торгової марки «Java»), вирішено перейменувати Java EE на Jakarta EE. Повернемося до книг. Тут все не так просто, як могла б здатися на перший погляд. По-перше, обрана тема звучить як «Кращі книги з Java EE» і тут невеликий нюанс, книги з фреймворків тут не підходять, а значить навіть «Вищий розум» з відповідей. Маїл тут не допоможе.
По-друге, виходячи з теплої та затишної зони комфорту під назвою «Java SE», варто забути про російську мову і перейти буржуазною.
По-третє, якщо на такому рівні ви не обзавелися ментором або друзями з числа хоча б трохи досвідченіших розробників, які могли б допомогти порадою і наставити на шлях істинний, то… все дуже сумно ^_^
Але ж ви не думали, що ці дрібниці зупинять мене? Отже, незважаючи на те, що платформа Java EE змінила семпай, за умовами передачі буде збережена свята святих Java з часів Swing – це зворотна сумісність, а значить, нам нічого не заважає використовувати вже написану літературу. Для початку трохи інформації з найнадійнішого та достовірнішого джерела:
На основі платформи SE побудована платформа EE, що включає все те, що є в SE + додаткові технології для створення Комерційного ПЗ, що надає API для роботи з нею, фреймворки працюють вже поверх неї. Дешево й сердито.
А також список технологій та посилання на специфікацію:
Навіщо тут ця історія? Давай вже найкращі книги з Java EE! Хвабонку, «нумерація Java» йде платформою SE, тобто. Java 10 - по суті Java SE 10, а як видно з скрин, у платформи EE все трохи інакше. Після тривалої перерви, сліз, страждань та істерик, після кількох гучних заяв про розрив відносин, Oracle під тиском громадськості все ж таки випустабо платформу Java EE8 – наприкінці серпня 2017 року, значить, найсвіжіша література російською мовою буде по платформі EE7, ось чим я керувався під час пошуку книг.
