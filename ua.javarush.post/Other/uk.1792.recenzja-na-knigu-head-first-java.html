Доброго дня, шановні джаваршівці та джаварашатеси! У процесі проходження курсу CodeGym так само почитаю рекомендовані книги. Рекомендована для прочитання книга № 1 - "Вивчаємо Java" Кеті Сьєрра, Берт Бейтс(Head First Java By Bert Bates, Kathy Sierra). Книга доступна російською мовою у паперовому та електронному вигляді. Щоправда, електронний вигляд це скан у PDF. Читабельний, але не кращої якості. Зараз я вже вивчаю 17 розділ книги з 18, тому склав про неї свою думку. Дещо і висловлю. До оцінки книги я підходитиму з методологічних позицій вивчення точних наук, викладених у книзі "Думай як математик" Барбари Оаклі (Ban Barbara Oakley PhD). Мені здається, що в цій книзі викладено найкращу в даний момент методику та прийоми вивчення точних наук. Отже, HF Java чудово підходить для вивчення Java з нуля. У книзі послідовно подаються всі основні теми. Пояснення прості та дохідливі. Повторюся, книга чудово підходить:
  • для вивчення Java з нуля та отримання загального уявлення про неї;
  • як матеріал, що використовується для з'ясування (розуміння) теми;
  • для повторення тим, розуміння яких зникло.
HF Java хороша для використання у зв'язці з CodeGym. Вона розкриває теми зрозуміліше JR, іноді дає додаткову (проти лекцій JR) інформацію з них. Крім цих позитивних, як я сподіваюся, чорт треба мати на увазі таке. Книжка максимально далека за стилем викладу від академічних підручників. Це може спричинити певний дискомфорт. Завдання засвоєння матеріалу дуже специфічні. Зокрема, взагалі відсутні завдання написання коду. Як наслідок, з кожної теми необхідно вигадувати завдання з написання відповідного коду самостійно. Доступність матеріалу для розуміння не тягне за собою автоматичну легкість його запам'ятовування. Необхідно прикладати додаткові значні зусилля як запам'ятовування вивчених концепцій, так запам'ятовування фактичної інформації. Я це роблю за допомогою Anki. Підсумок. З повагою, зелена жаба.
