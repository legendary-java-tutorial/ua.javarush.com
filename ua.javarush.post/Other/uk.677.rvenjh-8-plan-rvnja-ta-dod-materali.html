План рівня:
  • Колекції: LinkedList, HashSet, HashMap. Date – дата.
  • Колекції Set – безліч, List – список, Map – словник (або карта).
  • List -> ArrayList, LinkedList, Vector, Stack.
  • Set -> HashSet, TreeSet, SortedSet.
  • Map -> HashMap, TreeMap, SortedMap, Hashtable.
  • Відмінність реалізацій
  • Робота з ітератором Iterator. Методи hasNext(), next()
  • Map.Entry. Методи getKey(), getValue()
  • Робота з датою – клас Date.
