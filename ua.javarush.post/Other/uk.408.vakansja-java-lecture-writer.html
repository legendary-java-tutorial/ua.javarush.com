Чи любите писати статті або виступати на конференціях з доповідями з Java? Вчіть людей, ведете навчальні курси з Java або хочете створити свої? Або може бути пишіть книги про… Java? Тоді, можливо, вас зацікавить можливість приєднатися до CodeGym. Нам потрібна людина, яка вміє писати хороші навчальні лекції та статті з Java.
Що ми пропонуємо:
  • гарна можливість для самореалізації та професійного зростання;
  • адекватний менеджмент та цікаві завдання;
  • офіс з усіма зручностями (зона відпочинку, бібліотека, кухня, безкоштовний чай, кава, фрукти та печиво);
  • оплачувана відпустка (24 календарні дні на рік), оплачувані лікарняні, медстраховка;
  • зручне розташування офісу (ст.м. Либідська);
  • дружня команда професіоналів;
  • тім-білдінги та корпоративи за рахунок компанії;
  • можливість іноді працювати з дому.
Чим треба буде займатися:
  • писати навчальні лекції та статті на задані теми;
  • писати навчальні лекції з певного ТЗ;
  • писати лекції для користувачів певного профілю;
  • підбір правильних аналогій;
  • переписування статей із книг у вигляді навчальних лекцій;
  • переписування лекцій, що вибиваються із загального стилю;
  • написання нових лекцій згідно з ТЗ методиста;
  • складання питань (і відповідей) для підготовки до співбесіди;
  • писати мотиваційні статті;
  • писати статті про ІТ-галузь;
  • писати статті про професійне життя програмістів.
Вимоги до кандидата:
  • мінімум 1 року досвіду Java-програмістом;
  • мінімум 1 року досвіду викладання та навчання Java;
  • відмінне знання Java Core;
  • досвід роботи з Spring, Hibernate, Servlets;
  • досвід написання навчальних статей або книг.
Буде плюсом:
  • написання власних книг із програмування;
  • ведення власних навчальних курсів.
Портрет ідеального кандидата: Java-програміст з аналітичним складом розуму, ініціативний та відповідальний, якому подобається ділитися досвідом, навчати людей та допомагати новачкам.
