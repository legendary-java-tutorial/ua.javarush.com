----------------------------------------

Можливо, ви чули, що IntelliJ IDEA є клавіатурно-орієнтований IDE. Це означає, що ви можете працювати IntelliJ IDEA, не торкаючись миші. Це допоможе вам залишатися зосередженим і, зрештою, бути продуктивнішими. Саме тому важливо знати розкладки IntelliJ IDEA, тобто, які клавіші та комбінації клавіш, які дії викликають.
1. Налаштування
IntelliJ IDEA поставляється з набором клавіатурних розкладок, які були спеціально адаптовані для використання з різними операційними системами, а також полегшують перехід з інших IDE та редакторів. Наприклад, якщо ви тривалий час користувалися Eclipse, NetBeans або шанувальник Emacs, ви зможете знайти розкладку, яка підходить саме вам. Щоб вибрати розкладку, яку потрібно використовувати, перейдіть до меню Settings → Keymap. Будь-яка розкладка може бути додатково адаптована з урахуванням ваших звичок, тобто ви можете додати власні комбінації клавіш і змінити існуючі.
2. Пошук дії щодо клавіатурної комбінації
Якщо вам цікаво, яка дія викликає певне поєднання кнопок, подивіться на наступний малюнок. Повний довідник розкладок завжди доступний через Help → Default Keymap Reference (Розкладка за замовчуванням), а також на сайті IntelliJ IDEA.
3. Виклик дії на ім'я
Щоразу, коли вам необхідно виконати деякі дії, але ви не знаєте, яку комбінацію клавіш використовувати натисніть Shift + Ctrl + A (Shift + Cmd + A для Mac) безпосередньо з редактора і почніть друкувати те, що ви шукаєте.
4. Конфлікти з клавіатурними комбінаціями системи
Іноді, якщо ви вибрали розкладку, специфічну для вашої операційної системи, все ще може бути конфлікт між клавіатурними комбінаціями, що використовуються в IntelliJ IDEA та вашій ОС. Щоб уникнути цих конфліктів, просто використовуйте наступну хитрість у налаштуваннях системи.
5. Mac OS X
Переключіть F1 - F12 для стандартної поведінки функціональних клавіш, тому що всі вони активно використовуються в IntelliJ IDEA. Вимкнути Show Spotlight search field, тому що її клавіатурна комбінація Ctrl+Space – це комбінація за замовчуванням для завершення коду в IntelliJ IDEA. При першому запуску IntelliJ IDEA на Mac OS X вибрано стандартну розкладку для Mac OS, але ви можете вибрати Mac OS X 10.5+ розкладку, яка була розроблена спеціально для Mac OS X версії 10.5 і вище, розкладка більш дружня для користувачів, які звикли до клавіатурних комбінацій Mac OS X.
6. Ubuntu
Ось список хитрощів, які вам потрібно зробити з системою для клавіш за замовчуванням, так щоб вони не заважали діям IntelliJ IDEA (вказаних у дужках). Вимкніть тінь вікна, призначене на Ctrl + Alt + S (Діалог налаштувань) Змініть або відключіть блокування екрана, призначено на to Ctrl + Alt + L (Форматувати код) Змініть або відключіть запуск терміналу, призначене на Ctrl + Alt + T (Оточити вираз конструкцією ) Змініть або вимкніть, перемикання робочої області, призначене на Сtrl + Alt + Стрілки (Навігація) Вимкніть переміщення вікна, призначене на Alt + F7 (Пошук використання) Змініть або вимкніть зміну розміру вікна, призначене на Alt + F8 (Обчислення виразу) 7. Vim І, нарешті, якщо ви є відданим шанувальником Vim і не можете піти на кодування в іншому редакторі, ви будете раді дізнатися, що IntelliJ IDEA має плагін емуляції Vim . Щоб включити розкладку Vim в IntelliJ IDEA, вам необхідно завантажити і включити плагін IdeaVim в Settings → Plugins.
