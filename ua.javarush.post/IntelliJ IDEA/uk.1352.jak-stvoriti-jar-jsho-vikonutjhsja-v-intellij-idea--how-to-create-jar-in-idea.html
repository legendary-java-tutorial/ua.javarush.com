Створюємо виконуваний jar в Intellij IDEA

Як виявилося, не так просто знайти в просторах інтернету інформацію про це. Є багато посилань на те, як це робиться в Eclipse, а про Intellij IDEA сказано не набагато більше за нуль. Тому я створив цю посаду. IntelliJ IDEA дає можливість швидко створювати виконуваний JAR-файл вашої програми, що містить модулі з усіма залежностями. Для того, щоб створити JAR, необхідно виконати всього 3 пункти:
  1. Натисніть кнопку ' + ' у діалоговому вікні Project Structure та виберіть відповідний пункт (Малюнок 1):
    Малюнок 1 – Project Structure -> Artifacts -> Jar -> From modules with dependencies

  2. Далі IntelliJ IDEA показує діалог, що дозволяє налаштувати новий артефакт (Малюнок 2):
    Тут потрібно вибрати головний клас вашого проекту та натиснути ОK

    Рисунок 2 – Діалог створення нового артефакту JAR

  3. Після натискання кнопки ОК, ви можете створити файл Jar за допомогою пункту меню Build -> Build Artifact
    За замовчуванням всі бібліотеки будуть витягнуті в цільовий Jar. Сам файл, що виконується, буде згенерований в директорії out -> artifacts вашого проекту (Малюнок 3):

    Малюнок 3 – Розташування згенерованого JAR файлу

    Все, тепер виконуваний файл можна запускати.
