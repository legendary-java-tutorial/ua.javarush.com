Відкриття проекту JavaRushHomeWork

----------------------------------------

Ви завантажили архів з , розпакували його та бачите наступні дві папки: Папка містить шаблон проекту, в якому Ви вирішуватимете завдання. Папка містить плагін, який необхідно підключити до IntelliJ IDEA (Ідея). Відкриваєте Ідею. Ідіть у мен

----------------------------------------

1. Качаєте архів

Ви завантажабо архів з CodeGym , розпакували його та бачите наступні дві папки: Папка CodeGymHomeWork містить шаблон проекту, в якому Ви вирішуватимете завдання. Папка CodeGymIdeaPlugin містить плагін, який необхідно підключити до IntelliJ IDEA (Ідея).

2. Відкриваєте проект

Відкриваєте Ідею. Ідіть у меню File -> Open... . Відкриваєте шлях, куди розпакували архів. Клацніть на папці CodeGymHomeWork і тиснете кнопку OK .
Відкритий проект виглядає ось так:

3. Налаштовуєте проект

Відкриваєте параметри проекту File -> Project Structure... . Виставляєте Project language level на 7.0
У Modules також виставляєте Language level на 7.0 . Забираєте папку .idea з проекту, навіщо її потрібно вибрати та натиснути Excluded .
У SDKs, якщо там нічого немає, тиснете зверху зелений + , зі списку вибираєте JDK і вказуєте шлях до java . Тиснете OK . Увага! На курсі java7, ставте 7 версію, щоб не було проблем із налаштуваннями. Коли проект буде перекладено java8, ми повідомимо про це. Переконайтеся, що JDK home path вказує на шлях до вашої джави. У вкладці Classpath натискаєте зелений + праворуч. Підключаєте всі джарники з *шлях до джави*/jre/lib Тиснете кнопку Apply і перезавантажуєте проект.

4. Підключаєте плагін

Відкриваєте налаштування Ідеї File -> Settings... для Plugins і тиснете кнопку Install plugin from disk... .
Вибираєте шлях до плагіна IdeaPlugin.jar , який знаходиться в папці lib всередині CodeGymIdeaPlugin (див. п 1) і натискаєте кнопку OK.
Використовуєте нові налаштування натисканням на Apply і перевідкриваєте Ідею - тиснете Restart .

5. Тестуємо роботу плагіна

З'явабося дві нові кнопки:
  • натисніть ліву кнопку (смугасту) для отримання списку своїх завдань
  • натисніть праву кнопку (галочку), щоб відправити завдання на перевірку
Важлива інформація Файл CodeGymPlugin.properties містить секретний ключ, нікому його не показуйте. Ви завжди зможете залогінитись у свій аккаунт по ньому. Приємного навчання!
