Цей FAQ створений з питань IntelliJ IDEA в ВК топіці в нашій групі .

Як у idea зробити так, щоб запускався відкритий файл, а не старі Solution?

Запустити код можна, якщо є метод main. У цьому випадку є два способи:
  1. Натисніть Ctrl+Shift+F10.
  2. Натисніть правою кнопкою миші по робочій області у потрібному класі та виберіть Run або Debug

Як зробити, щоб на екрані відображався номер позиції в рядку коду, в якому знаходиться курсор?

Відкриваємо панель View > Status Bar ставимо галочку. Якщо галочка стоїть, то зробіть менше вікно, можливо, у taskbar перекриває вікно IDEA.

Видає помилку:

java: try-with-resources is not supported in -source 1.6
(use -source 7 or higher to enable try-with-resources)
Потрібно виставити рівень джави вище 6 версії. Відкриваємо Project Structure ctrl+alt+shift+s > Project > Project language level .

У лекціях є завдання, а плагін їх не бачить

Схоже, що плагін у IntelliJ IDEA бачить завдання іншого облікового запису на сайті. Потрібно порівняти секретні ключ на сайті та плагіні. На сайті секретний ключ можна знайти в налаштуваннях . У плагіні секретний ключ знаходиться у файлі CodeGymPlugin.properties

Як в intellij запустити програму в покроковому режимі? Щоб, наприклад, при виконанні циклу від 0 до 4 консолі виводабося присвоєні значення і результати обчислень на кожному кроці

Читайте статтю "Як користуватися Debug у Intellij IDEA".

Неналежним для find package java.lang в classpath or bootclasspath

Відкриваємо панель File > Project Structure > Platform Settings > SDKs У вкладці Classpath підключіть усі jar файли з "шлях до встановленої джави" /jre/lib .

Чому IntelliJ IDEA закреслила методи, які чудово працюють?

Це застарілі методи, якими небажано скористатися. Вони можуть бути прибрані у наступних версіях мови. Ці методи позначені анотацією Deprecated , і IDEA попереджає про це.

Де в IntelliJ IDEA вводити дані, які читатимуться у програмі

Механіка дій:
  1. Пишемо програму.
  2. Запускаємо Run
  3. Унизу з'являється консоль.
  4. Пишемо дані в консоль
  5. Тиснемо Enter

Як повернути віконце з результатом компіляції програми, що знаходиться знизу

Використовуємо гарячі клавіші: Alt+0- Відкриється панель Message Alt+4 - Відкриється панель Run Інші панелі можна знайти View > Tool Windows .

Виводить ...the SDK is not specified... Або код розфарбований червоним кольором

У проекті необхідно вказати шлях до SDK.
  1. Відкриває панель File > Project Structure (ctrl+alt+shift+s).
  2. Вибираємо вкладку SDKs .
  3. Вказуємо шлях до встановленого пакету JDK .
Якщо код все ще підфарбований червоним, але програма працює, виконайте File > Invalidate Caches

Чому IDEA виділяє мені деякі елементи коду коричневим тлом?

Це так званий Inspections. Можна вважати це як Code Review . Якщо навести на такий елемент курсор ведмедика, IDEA видасть підказку і автоматично підмінить код більш правильний на думку IDEA. У ситуації з " for " буде заміна на більш короткий " for each ".

Як мені включити стиль відступів K&R?

Використовуємо гарячі кнопки: Ctrl + Alt + L. Докладніше у статті "Стиль та форматування коду".

Де в опціях проставити, щоб результат виконання завдання на екран виводився, а чи не зберігався в консоль і файл відповідно?

Параметр у вікні Run/Debug Configurations був змінений .

  1. У верхньому лівому куті екрана є список, що випадає (зліва від зеленого трикутника). Натискаємо та вибираємо Edit configurations .

  2. Потім вибираємо вкладку Logs . Забираємо галочку навпроти Save console output to file .

Після переустановки IntelliJ IDEA та плагіна чи збережеться прогрес та підписка?

Ваш прогрес та передплата прив'язані до секретного ключа (файл CodeGymPlugin.properties ). Жодних проблем не буде. Якщо хочете зберегти всі рішення – просто не видаляйте проект, а решту можете сміливо встановлювати заново.

Після компіляції замість кирабоці показує питання або незрозумілі символи, в чому може бути причина?

Проблема з кодуванням. Проблеми виникають коли виведення в UTF8, а введення в WIN1251 (наприклад).

  1. Відриваємо панель File > Settings , у рядку пошуку вводимо Encoding . Виставляємо Project Encoding - UTF-8 .

  2. Ділам рестарт і чистимо кеш IDEA . Панель File > Invalidate Caches/Restart

Для тих, хто знає англійську, є гарна стаття на тему кодування

Як зробити, щоб під час запуску IntelliJ IDEA відразу відображалися номери рядків коду?

Нумерацію рядків можна увімкнути двома способами:

  1. Правий клік миші по порожній ліворуч від коду (див. скрін нижче) > Show line numbers .

  2. Відкриваємо панель File > Settings > Editor > General > Appearance. Налаштувати галку Show line numbers.

Що потрібно зробити, щоб працювало виділення shift-ом? Не працює ctrl+c та ctrl+v

Схоже, що встановлено плагін Vim. Потрібно відключити плагін:

  1. File > Settings (Ctrl+Alt+S)

  2. Plugins > Search -> vim (у пошуку)

  3. Забрати галку. Ok. Restart .
