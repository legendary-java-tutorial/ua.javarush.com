Пропонуємо до вашої уваги поради щодо підготовки до сертифікації, опубліковані Software Architect компанії Infotech Джейвіном Полом.
Пройшло вже більше трьох років з 17 березня 2014 р., появи Java SE 8 і вже майже три роки всі бажаючі можуть скласти сертифікаційний іспит OCAJP8. Напевно, ви в курсі: щоб стати сертифікованим розробником Java SE 8, необхідно скласти два іспити – OCAJP8 (код іспиту 1Z0-808) та OCPJP8 (код іспиту 1Z0-809). Перший називається сертифікацією базового рівня (associate level certification), а другий – сертифікацією професійного рівня (professional level certification). Коли OCAJP8 тільки-но з'явився, не існувало досить хороших книг, присвячених підготовці до нього, так що багатьом кандидатам доводилося покладатися в основному на книги з Java 8та професійні симулятори іспиту, наприклад, Whizlabs та Enthuware. Сьогодні ж у тих, хто складає іспит OCAJP8 такої проблеми немає. У цій статті ви познайомитеся з декількома найкращими книгами для підготовки до іспиту OCAJP8.

Як підходити до складання іспиту

Правила успіху на іспиті OCAJP8 дуже прості: знайдіть хороший підручник (наприклад, будь-яку з наведених нижче книг), пишіть код щодня, проходьте якомога більше пробних іспитів. Достатньо бути чесним із самим собою при написанні коду: не тільки писати, а й розуміти, що ви пишете, та експериментувати з кодом. При відповіді на тренувальні питання із пробних іспитів уважно вивчайте всі варіанти відповідей. Важливо не тільки знати, яка відповідь правильна, але й розуміти, чому всі інші неправильні. І, нарешті, обов'язково читайте пояснення до тренувальних питань та пробних іспитів. Це найбагатше джерело інформації саме з цього іспиту. Крім того, вони допоможуть вам виправити свої помилкові уявлення та заповнити прогалини у ваших знаннях. Коли я отримав, ще за часів Java 1.4, свою першу сертифікацію, саме ці три правила допомогли мені досягти високого балу, тож відтоді я завжди рекомендую цей підхід своїм читачам. Жодних срібних куль – лише чесна підготовка з використанням правильної навчальної літератури, книг та тренувальних питань.

Кращі книги по OCAJP8

Хоча у платформі Java SE 8 з'явилося кілька нових концепцій, наприклад, лямбда-вирази, посилання на методи, потоки даних, нові API для роботи з датами та часом та ін., великих відмінностей від OCAJP7 у ній немає. Так що якщо у вас вже є якісь книги з OCAJP7 , ви цілком можете скористатися ними для підготовки, а потім актуалізувати свої знання відповідно до вимог нового іспиту. Якщо ви новачок або отримуєте свою першу сертифікацію з мови Java, можете використовувати для підготовки до екзамену OCAJP8 з кодом 1Z0-808 наступні книги. Пам'ятайте, що крім різних версій Java (7 і 8) є ще два різні іспити - OCAJP8 і OCPJP8 - так що для вибору потрібних книг обов'язково потрібно враховувати код іспиту.

OCA Java SE 8 Programmer I Certification Guide

Автор книги звати Мала Гупта (Mala Gupta). Я з нетерпінням чекав виходу цієї книги з того часу, як прочитав посібник із сертифікації OCAJP7 того ж автора. І вона, безумовно, чудово доповнила його новими можливостями Java 8. Ви знайдете в цій книзі масу корисних схем, нотаток на полях, а також стислий і зрозумілий виклад різних граничних випадків, важливих для отримання сертифікації Java. Як завжди з подібними підручниками, ця книга охоплює всі необхідні для іспиту OCA Java SE 8 Programmer I Certification (код екзамену 1Z0-808) питання, включаючи надбудови мови Java, наприклад, створення та використання лямбда-виразів, нові API для роботи з датами та часом та ін. Вона також включає повномасштабний пробний іспит, який можна використовувати разомз сертифікаційними питаннями з Java 8 Девіда Майєра . Навіть якщо ви добре знайомі з Java 8, впевнений, що ви зможете знайти в цій книзі щось нове для себе або дізнатися про нові факти про вже знайомі вам речі. Загалом це відмінна книга для Java-програмістів усіх рівнів, що готуються до сертифікаційного іспиту OCAJP8, тому, якщо хочете купити для підготовки лише одну книгу – купуйте саме її.

OCA: Oracle Certified Associate Java SE 8 Programmer I Study Guide: Exam 1Z0-808

Ще одна чудова книга для підготовки до іспиту OCAJP8. Це найбільш актуальний та повний виклад питань OCAJP8. Написали її Жанна Боярскі (Jeanne Boyarsky) та Скотт Зелікофф (Scott Selikoff), які проробабо чудову роботу з охоплення всіх важливих нюансів іспиту OCAJP8. Книга починається з тесту для перевірки своїх знань і містить безліч прикладів вивчення як нових, так і старих принципів мови Java. Ще одна корисна частина цієї книги – докладні пояснення щодо питань вищезгаданого тесту. Як і симулятор іспиту OCAJP 8 від компанії Whizlabs , вона не тільки показує, чому відповідь A або B – правильна, але й пояснює, чому інші відповіді помилкові. Це дуже корисно для розуміння принципів роботи та виправлення своїх хибних уявлень.

OCA Java SE 8 Programmer I Study Guide (Exam 1Z0-808)

Це ще одна книга, присвячена екзамену OCAJP8, що містить відомості з першоджерела, тобто компанії Oracle. 3-тє видання цієї, написаної Едвардом Фінеганом (Edward Finegan) і Робертом Лігуорі (Robert Liguori), книги було повністю перероблено та містить актуальну та відповідну темам іспиту інформацію. Подібно до попередньої книги, вона містить безліч вправ і тестів для перевірки своїх знань. До неї також додається компакт-диск, що включає два повномасштабні випробувальні іспити. Хоча рівень питань відрізняється від реального іспиту, і краще скористатися симулятором іспиту OCAJP 8 компанії Whizlabs, проте на них можна непогано попрактикуватися.

Oracle Java SE8 Programmer Exam Self-Practice Review Questions for Exam 1z0-808

Ця книга призначена для опрацювання питань іспиту OCAJP8 (код 1Z0-808). Вона містить понад 120 питань для тренування. Її можна використовувати, заради підвищення якості підготовки, разом із будь-якою з вищезгаданих книг. Ці тренувальні питання можна використовувати разом із моїм списком безкоштовних пробних іспитів OCAJP8 , щоб набратися досвіду перед тим, як перейти до фінальної стадії підготовки у вигляді пробного іспиту на симуляторах іспитів від компаній Enthuware або Whizlabls.
