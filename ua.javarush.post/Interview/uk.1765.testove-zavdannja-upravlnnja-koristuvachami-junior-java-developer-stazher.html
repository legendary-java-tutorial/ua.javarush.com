----------------------------------------

Необхідно розробити веб-додаток, що дозволяє керувати користувачами (створювати, редагувати, переглядати список та деталі, видаляти)
Функціональні вимоги до веб-додатку:
  1. Наявність RESTful інтерфейсу;
  2. Підтримка операцій управління (створення, читання, редагування та видалення) користувачами.
    Користувач визначається такими значеннями:
    • Ім'я
    • Прізвище
    • дата народження
    • Логін
    • Пароль
    • Поле введення “Про себе”
    • Адреса проживання
  3. Наявність користувальницького WEB-інтерфейсу
Нефункціональні вимоги:
  • JEE стек технологій
  • Фреймворки, бібліотеки: Hibernate , Spring
  • Система зберігання даних: PostgreSQL
  • Код програми необхідно забезпечити коментарями
  • Додаток повинен збиратися за допомогою maven без встановлення або налаштування будь-яких додаткових компонентів;
  • Архів з результатом тестового завдання повинен містити текстовий файл readme.txt з інструкцією зі збирання, налаштування, конфігурування та розгортання програми (якщо необхідно).
