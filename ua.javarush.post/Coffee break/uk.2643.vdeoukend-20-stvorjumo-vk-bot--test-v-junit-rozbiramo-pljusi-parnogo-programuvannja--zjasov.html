Java | Робимо VK-бота, який пише новини ігор з Steam API

Хочете попрактикуватися в IntelliJ IDEA, але не знаєте, який проект краще вибрати? Давайте подивимося, як за 7 хвабон можна створити VK-бот, який надсилає ігрові новини з платформи Steam. Все, що для цього знадобиться – дві бібліотеки: VK Bot Java SDK та Steam Web API for Java. Отримавши навички створення ботів для Вконтакте, ви можете спробувати зробити самостійний проект для інших соцмереж: Facebook або Twitter. Вміння створювати автоматизованих роботів стане додатковим плюсом для будь-якого новачка Java-програмування.

Парне програмування

Парне програмування - один із ефективних способів покращити якість свого коду. Адже, крім роботи в парі, кодери також і перевіряють результати роботи один одного. Завдяки цьому можна не вдаватися до допомоги сторонніх фахівців та якісніше розподіляти ресурси компанії, що особливо важливо для невеликих команд. Адже, як відомо, на написання коду звичайний програміст витрачає приблизно 50% свого робочого дня. У представленому тут відеоролику своє ставлення до парного програмування пояснює блогер і програміст Михайло Фленов — колишній автор журналу «Хакер» (рубрики Кодинг та FAQ), творець кількох книг про розробку та кібербезпеку.

Створення простого тесту JUnit

Якщо раптом ви недавно почали практикуватися в IDE Eclipse, вас напевно має зацікавити наочний приклад покрокового створення тесту JUnit. Завдяки бібліотеці JUnit розробник може ефективно проводити модульне тестування проектів на Java будь-якої складності. Незважаючи на те, що автор відео вибрав для тесту одну з найпростіших програм, «Калькулятор», за бажання ви можете повторити алгоритм його дій на будь-якому іншому проекті. Важливо лише правильно розпочати та закріпити основи на конкретних прикладах.
