5 інструментів, які покращать вашу роботу з GitHub

Джерело: Medium GitHub – найпопулярніший сервіс командної розробки. На ньому можна знайти безліч громадських репозиторіїв різними мовами програмування. Ви можете використовувати GitHub, щоб зберігати свій код, працювати з колегами за командою та іншими розробниками. Також цей сервіс можна використовувати з навчальною метою: вивчати чужий код, популярні бібліотеки з відкритим кодом і брати участь у їх розробці. Крім того, ви завжди можете удосконалити GitHub, «прокачавши» та налаштувавши його під себе. Представляємо вам список інструментів, які можуть допомогти.

Awesome Autocomplete for GitHub

Одна з найкорисніших фішок будь-якого пошуку – система автодоповнення пошукового запиту. І річ навіть не в тому, що так ви можете заощадити час і не набирати весь свій запит повністю. Головне, що тут вам пропонуються варіанти запитів. Як ви вже здогадалися, за допомогою розширення браузера Awesome Autocomplete for GitHub можна отримати додаток у рядку пошуку на сайті GitHub. Його можна використовувати для пошуку як по репозиторіям, так і по облікових записах людей і організацій.

GitHunt

У функціоналі GitHub є одна цікава нагода: там можна побачити список трендових проектів. Тобто ви можете ознайомитися з новими інструментами та бібліотеками, які останнім часом привернули до себе особливу увагу та отримали багато зірок від інших розробників. Щоб полегшити пошук нових трендів, використовуйте розширення браузера GitHunt. Воно дозволяє переглядати трендові проекти у нових вкладках з конкретної мови програмування за день, тиждень чи місяць.

GitHub Notable Comments

Один з найкорисніших розділів GitHub, де можна знайти рекомендації, поради чи прохання щодо допомоги, що стосуються конкретного проекту. У ньому розробники можуть скаржитися на баги, пропонувати рішення чи висловлювати ідеї, що, на думку, слід включити у проект. Якщо ви хочете швидше переміщатися за коментарями та читати тільки найкорисніші, скористайтеся розширенням браузера GitHub Notable Comments. Встановивши його, зайдіть у вкладку Issues потрібного проекту та подивіться на його вміст. Для прикладу можна взяти Issueпроекту Rails. Там ви знайдете 66 коментарів, але вам, природно, хотілося б прочитати найкорисніші. Допоможе цьому невелика бічна панель, яка з'являється після встановлення розширення. Так ви можете моментально перейти до коментаря, який отримав найбільше тих чи інших реакцій.

Enhanced GitHub

Встановивши розширення Enhanced GitHub у своєму браузері Google Chrome, ви зможете моментально отримати посилання на скачування не тільки всього репозиторію, але й будь-якого файлу в ньому. Також ви зможете копіювати вміст файлів. Хоча при копіюванні коду втрачається розмітка, але це досить корисний інструмент.

Git First

Якщо ви раптом захочете подивитися свій перший коміт у початому вами проекті або перший коміт вашого колеги, скористайтесь плагіном Git First. Так ви суттєво скоротите час на пошуки. Потрібно лише встановити його в браузері, після чого у вас у контекстному меню (на правому кліку миші) з'явиться додатковий пункт: Initial Commit. Дуже зручний інструмент, який дозволяє швидко переглядати, як інші люди розпочинають свої проекти.
