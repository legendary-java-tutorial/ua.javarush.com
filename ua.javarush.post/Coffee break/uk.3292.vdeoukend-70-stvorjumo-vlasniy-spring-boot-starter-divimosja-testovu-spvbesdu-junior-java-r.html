Java Tech Talk: Hand-made Spring Boot Starter

Spring Boot Starters - це набір дескрипторів залежностей, які ви можете підключити до своєї програми. Використовуючи Spring Boot Starters, розробник позбавляє себе необхідності копатися в прикладах коду, копіюючи і вставляючи безліч дескрипторів залежностей. Boot Starters заощаджує час на конфігурації та дозволяє зосередитися безпосередньо на проектуванні програми. Єдиним недоліком цього інструменту є те, що інколи starter не надає розробнику потрібного йому функціоналу. Однак і цю проблему цілком можна вирішити, адже завжди можна створити власний starter. Нещодавно стриманий на каналі Minsk Java Community присвячений цьому завданню. Як реалізувати з нуля свій Spring Boot Starter покаже та розповість Тимофій Зубков.

Що таке легаси код? Моноліт vs легаси vs мікросервіси. Варіанти архітектур

Ви ризикуєте потрапити в незручну ситуацію на співбесіді, якщо досі не знаєте різницю між монолітом і легасом кодом. Крім того, вам знадобляться знання про те, чим обидва ці поняття відрізняються від мікросервісної архітектури. Розібратися з цією термінологією допоможе Олексій Корольов: ви дізнаєтеся про особливості роботи з монолітом і легасом кодом, а також багато іншої корисної інформації — наприклад, про недоліки популярної технології мікросервісів.
