Три способи використовувати DOCKER з JAVA

Навряд чи хтось візьметься заперечувати твердження, що Docker — це найкращий спосіб «упакувати» додаток у контейнер. Вже потім ви можете запускати контейнер у будь-якому необхідному вам середовищі: у тому числі, на віртуальному сервері або у хмарі. Існує щонайменше три способи використання Docker для Java проекту. Про те, як з ними працювати сьогодні ви дізнаєтеся з відео, підготовленому для вас автором YouTube-каналу Codelike. Також вас ознайомлять з перевагами та недоліками кожного методу роботи з Docker. Всі способи, про які йтиметься, взяті на прикладі Spring Boot.

- Хто такий Helidon і що з ним можна робити?

Дмитро Корнілов, Java-розробник із празького офісу Oracle, представив доповідь, присвячену Helidon – популярному набору Java-бібліотек для створення мікросервісів. Переглянувши відеоролик, ви дізнаєтеся, що таке Helidon, як він працює у системі сучасних засобів розробки мікросервісів і чим від них відрізняється. Частина доповіді Дмитра присвячена реактивному програмуванню, підтримці MicroProfile та GraalVM, сучасним засобам трейсингу та моніторингу. Крім того, глядачі побачать демо з побудовою двох RESTful сервісів за допомогою реактивного програмування та MicroProfile.

9 способів прискорити пошук роботи програмісту-початківцю

Якщо ви вважаєте, що вже підійшли до етапу пошуку своєї першої роботи як розробник, тоді вам напевно припаде до душі відео з дев'ятьма «підказками» від Антоніни Листопадової. Свій відеоролик дівчина присвятила найбільш ефективним, на її погляд, порадам, як прискорити пошук роботи програмісту-початківцю. Ймовірно, деякі рекомендації, які озвучила Тоня, вам уже знайомі, але решту порад ви почуєте вперше. Відео принесе користь кожному, хто має намір шукати роботу або має проблеми з проходженням співбесід.
