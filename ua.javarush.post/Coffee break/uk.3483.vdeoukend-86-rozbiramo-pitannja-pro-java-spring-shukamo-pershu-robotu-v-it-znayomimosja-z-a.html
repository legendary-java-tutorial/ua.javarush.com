Співбесіда Java Spring — Розбір запитань із співбесід на Java Developer

У відеоролику зібрали найцікавіші питання, які ставлять на технічних співбесідах Java-розробника. Відео складається із двох частин. У першій розповідається про Java Collections, Hash Code & Equals Contract, Functional interfaces, Stream API, Garbage collector, пристрій пам'яті, heap, concurrency. У другій частині відео ви дізнаєтеся про Spring: Spring Beans, Spring Data Repository, Анотації, Programmatic TM та Spring MVC.

Де програміст шукати першу роботу в IT

Пошук роботи — проблема будь-якого новачка, який завершив навчання мови програмування. Проте відсутність досвіду — аж ніяк не вирок! Розробник-початківець має багато способів знайти першу роботу в IT, навіть якщо до цього він ніде не працював за своєю новою спеціальністю. У цьому відео блогер Sweet Coder ділиться корисними рекомендаціями щодо пошуку роботи, багато з яких він особисто застосовував при пошуку роботи.

Автоматизація тестування - марна трата часу або "таблетка" від усіх хвороб?

Навіщо потрібна автоматизація тестування, у яких випадках її варто застосовувати, як вона відбувається, та в чому її переваги для розробника? Ви познайомитеся з конкретними прикладами, як автоматизація тестування прискорює процес роботи, і у яких випадках краще обійтися ручними тестами. Крім того, в цьому відео є багато корисної інформації про тестування додатків.

Хто такий і чим займається fullstack-розробник

Не знаєте, яку спеціалізацію у розробці краще вибрати: бекенд чи фронтенд? Чи, може, фулстек? У цьому відео ви дізнаєтеся, в чому полягають професійні обов'язки fullstack-розробників і чим вони відрізняються від звичних, розділених на окремі напрямки варіантів розробки додатків та сайтів. Відео можна переглянути за посиланням.
